#1 Game Development Game Development არის პროცესი, რომელიც მოიცავს ვიდეო თამაშების შექმნას და განვითარებას. ის მოიცავს თამაშის იდეის კონცეფციას, დიზაინს, სცენარს, პროგრამირებას, ანიმაციას, ვიზუალურ ეფექტებს, ხმის დიზაინს და ტესტირებას. პროცესში შეიძლება გამოიყენონ სხვადასხვა პროგრამირების ენები და სპეციალური game engines, როგორებიცაა Unity ან Unreal Engine, რათა შექმნან თამაშები კომპიუტერებისთვის, კონსოლებისთვის ან მობილური მოწყობილობებისთვის. AI (ხელოვნური ინტელექტი) AI არის კომპიუტერული სისტემების განვითარება, რომელიც ცდილობს ადამიანის მსგავსად იმოქმედოს და იფიქროს. ეს ტექნოლოგია მოიცავს მანქანურ სწავლას, ნერვულ ქსელებს, მონაცემთა ანალიზს და ალგორითმების გამოყენებას, რომ შეძლოს დადგენა, გადაწყვეტილების მიღება და პროგნოზირება. AI იყენებენ მრავალ სფეროში, დაწყებული სათამაშოებიდან და ვებ-პროგრამებიდან, დამთავრებული ჯანმრთელობის დაცვით და ფინანსურ სექტორში. Web Development Web Development არის ვებ-გვერდების და ვებ-აპლიკაციების შექმნის პროცესი. ის მოიცავს სხვადასხვა ტექნოლოგიებისა და პროგრამირების ენების გამოყენებას, როგორებიცაა HTML, CSS, JavaScript, PHP, Python, ან React, რათა შექმნან ვებსაიტები, რომლებიც მომხმარებლებს თავიანთი საჭიროებების შესაბამისად ემსახურება. Web Development შეიძლება დაიყოს ორ ძირითად ნაწილად: Front-End Development (მომხმარებლისთვის ხილული ნაწილი, დიზაინი და ინტერფეისი) და Back-End Development (სერვერის მხარე, მონაცემთა ბაზები და აპლიკაციის ლოგიკა). #2 Front-end Front-end არის ვებ-დეველოპმენტის ნაწილი, რომელიც მოიცავს ვებსაიტის ან ვებ-აპლიკაციის იმ ნაწილს, რომელიც მომხმარებლებს პირდაპირ ხილული და ხელმისაწვდომია. ეს არის ინტერფეისი, რომელთანაც მომხმარებელი ინტერაქციაში შედის. Front-end დეველოპმენტი იყენებს ისეთ ტექნოლოგიებს, როგორიცაა: HTML (HyperText Markup Language) — გვერდის სტრუქტურის შექმნისთვის. CSS (Cascading Style Sheets) — გვერდის დიზაინისა და სტილის დასაგეგმად. JavaScript — ფუნქციონალურობისა და ინტერაქციების დასამატებლად. Front-end დეველოპერი მუშაობს, რათა ვებ-გვერდი იყოს ვიზუალურად მიმზიდველი, მარტივად გამოსაყენებელი და რეაგირებადი სხვადასხვა მოწყობილობებზე (მაგ., კომპიუტერზე, ტაბლეტზე, მობილურ ტელეფონზე). Back-end Back-end არის ვებ-დეველოპმენტის ნაწილი, რომელიც მოიცავს ვებ-გვერდის ან აპლიკაციის "სერვერის მხარეს" - იმ ნაწილს, რომელიც მომხმარებლისთვის უხილავია. ეს არის სისტემის ფუნდამენტური ნაწილი, რომელიც უზრუნველყოფს მონაცემების დამუშავებას, შენახვას და სერვერისა და მონაცემთა ბაზების მართვას. Back-end დეველოპმენტი იყენებს ისეთ ტექნოლოგიებს, როგორიცაა: სერვერის პროგრამირების ენები (მაგ., Python, PHP, Ruby, Java, Node.js) — სერვერზე ლოგიკის დასაწერად. მონაცემთა ბაზები (მაგ., MySQL, PostgreSQL, MongoDB) — მონაცემების შესანახად და მართვისთვის. API-ები (Application Programming Interface) — სხვადასხვა სისტემებსა და აპლიკაციებს შორის მონაცემთა გაცვლისთვის. Back-end დეველოპერი მუშაობს, რომ ვებ-გვერდის ან აპლიკაციის ლოგიკა იყოს მძლავრი და უსაფრთხო, რომ მომხმარებლებმა მიიღონ სწრაფი და სტაბილური გამოცდილება. მოკლედ რომ ვთქვათ, Front-end არის ის, რასაც მომხმარებელი ხედავს და იყენებს, ხოლო Back-end არის ის, რაც მართავს ყველაფერს "კულისებს მიღმა".
